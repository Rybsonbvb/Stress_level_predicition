# Stress Level Prediction — Machine Learning Project

## Overview

This project explores machine learning methods for predicting **stress level** based on behavioral, lifestyle, and health-related features.

The goal is to build classification models capable of predicting stress level from input variables using data analysis, preprocessing, and multiple machine learning algorithms.

The project demonstrates a complete data science workflow including:

* exploratory data analysis (EDA)
* data preprocessing and cleaning
* feature encoding and scaling
* model training and comparison
* performance evaluation

---

## Dataset

The dataset contains behavioral and lifestyle features related to stress indicators.

Target variable:

* **stress level** — predicted using supervised machine learning.

Features include variables related to:

* daily habits
* health indicators
* behavioral attributes
* lifestyle factors

*(Dataset not included in repository.)*
